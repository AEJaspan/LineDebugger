## How To

### Setup

```bash
pip install poetry
poetry install
```
### To Run

Use the VS code command to run and build a task. On Windows this is

```bash
ctrl + shift + B
```

Or via the VS code options panel, go:

```
Terminal -> Run Build Task
```
